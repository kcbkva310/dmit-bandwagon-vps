## DMIT 是什么，Pro 系列又是什么

DMIT 是 2018 年前后由华人留学生在纽约注册的 VPS 商家，目前运营洛杉矶、香港、东京三个数据中心。在国内玩 VPS 的圈子里，它的口碑出了名的两极：要么被人奉为 CN2 GIA 的天花板，要么被人嫌贵。

它的产品线分三档，逻辑很清晰：

- **Premium（Pro 系列）**：旗舰档。电信走 CN2 GIA，联通走 AS9929，移动走 CMI，三网双向优化。适合对延迟和稳定性有硬需求的用户。
- **Eyeball（EB 系列）**：中档。洛杉矶版回程走 CMIN2，香港版走三网 CMI，价格和 Pro 接近，流量给得更多，联通移动用户性价比更高。
- **Tier 1（T1 系列）**：经济档，国际线路，无中国大陆专属优化，流量大、价格低，适合海外业务或者做落地节点。

今天的核心话题是 **Pro 系列**，也就是 DMIT 的拳头产品。

## 搬瓦工对应产品怎么回事

搬瓦工（BandwagonHost）是 DMIT 的上游之一，这个说法在圈子里流传很久，基本属实。两家都有 CN2 GIA，但产品策略完全不同。

搬瓦工的优化套餐目前分几条主线：

**CN2 GIA ECOMMERCE（多机房版）**：一个套餐支持在十几个节点之间随时切换，包括洛杉矶 DC6/DC9、圣何塞、大阪、东京等。这是搬瓦工最大的差异点。入门款季付约 $65.89，年付 $169.99 起。

**日本大阪 / 东京 CN2 GIA（固定单机房）**：不支持换节点，但日本机房对大陆延迟更低，大阪是目前单机房里价格最低的，$49.99/月起，东京和香港同规格 $89.99/月起。

**香港 CN2 GIA（Ultra HK）**：延迟最低，价格最贵，和 DMIT 香港 Pro 是同一量级的定位，$89.99/月起。

搬瓦工有一个标志性机制：ECOMMERCE 系列遇到 IP 被封，可以直接在 KiwiVM 控制面板里换机房，相当于换了一台机器，不需要额外付费，也不需要等退款。这在 DMIT 那边是做不到的——DMIT 对应的方案是每 15 天免费换一次 IP，机房不变。

两种策略针对不同的痛点：怕 IP 被封就换节点的选搬瓦工 ECOMMERCE；更在意原始线路质量、偶尔 IP 出问题换个 IP 就行的选 DMIT。

## 实际使用下来的差异感受

DMIT Pro 晚高峰稳定性这块，用过的人通常的评价是"延迟数字没有大幅波动"。洛杉矶到国内三网晚高峰延迟大概在 150–180ms 浮动，丢包率极低，没有那种白天测速漂亮、晚上直接抽风的情况。

搬瓦工的线路稳定性评价历来偏好，但它的 ECOMMERCE 套餐里有一个小细节值得注意：500GB 或 1TB 的流量是按账单周期算的，季付套餐每季度重置，不是每月重置。初次买的时候容易误算。

DMIT 有一点确实不方便：工单系统为主，没有实时中文在线客服，深夜发工单有时响应慢。搬瓦工这块做得稍微成熟一些，工单一般几小时内有回复，文档体系也更完整。

## DMIT Pro 系列常见问题

**Q：DMIT Pro 套餐经常缺货，该怎么办？**  
洛杉矶 Pro 系列的热门规格（TINY、MINI）确实经常缺货，不定期补货，有时候几小时就售完。建议直接盯着官网库存页面，看到有货就下手，不要拖。香港和东京 Pro 也有同样问题。如果实在等不到，洛杉矶 EB 系列（CMIN2 回程）是个退而求其次的选择，联通移动用户用 EB 系列性价比其实更高。

**Q：DMIT Pro 的退款政策是什么？**  
3 天内流量未超过 30GB，可以全额退款。超过 3 天但在 1 个月内，按剩余使用价值退，会扣除支付网关手续费。也就是说不是无条件退款，下单前想清楚再买。

**Q：DMIT Pro 和搬瓦工 CN2 GIA 线路质量差多少？**  
实测数据来看，两家都走 CN2 GIA，晚高峰延迟数字差距不大。搬瓦工在零丢包和稳定性上有轻微优势，但 DMIT 的硬件更新（AMD EPYC 9005 跑分明显更高），磁盘 IO 实测也快不少。用户口碑普遍是"DMIT 线路够用，搬瓦工略稳，但 DMIT 便宜一截"。

**Q：DMIT 有官方优惠码吗？**  
有。洛杉矶 EB 系列有循环优惠码，季付及以上可以打折。东京 T1 系列有官方折扣码，季付享 7 折循环折扣。购买时在结算页面的 Promo Code 输入框验证是否有效，月付套餐大多不适用，需要季付或年付才能激活。

**Q：DMIT 流量超额会怎样？**  
不关机，限速。Pro 系列超额后限速到 1Gbps，T1 系列超额后限速到 50–100Mbps（WEE 套餐），继续用，不会直接断线。对个人用户来说基本不用担心流量超额的问题。
